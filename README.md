# OkulProjeleri İnternet Sitesi

Basit bir Next.js (App Router) iskeleti. Tailwind ile hızlı stil desteği için hazır.

Kurulum (PowerShell):

```powershell
npm install
npm run dev
```

Notlar:
- Gerekirse `npx tailwindcss init -p` ile Tailwind config/postcss dosyalarını güncelleyin.
- Bu proje basit bir şablondur; gerçek projede erişilebilirlik ve performans eklemelisiniz.
